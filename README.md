# agent-skills
Skills for AI coding agents to integrate with the solscan (pro-apis,...).

Skills follow the [Agent Skills](https://agentskills.io/) format.

## Available Skills

### Solscan Data

Helps agents integrate with the whole Solscan Data API. 

#### Installation

```bash
npx skills add solscanofficial/agent-skills --skill "solscan-data"
```
